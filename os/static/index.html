Organic Start Design

Colors

The primary color is moss (technically olive) and the secondary color is mustard (technically amber).
All of the colors including gray are extracted directly from the Organic Start logo and color variations are automatically generated.
Colors can be used with text- or bg-. There are 10 positive and negative shades. It's recommended to use text-white with positive shades.

Typography

Fluid Typography is always enabled. Always reference the type scale and understand typography.

Headers

The base font family is 'Open Sans' and 'Nunito' for headers and subheaders.
Use the display class or display-# classes to use our display (cursive) font which is 'Fredoka One'.

Gradients

Use any of over 180 gradient from WebGradients as a class thanks to @itmeo/webgradients. Substitute spaces with underscores. For example: Malibu Beach is malibu_beach
